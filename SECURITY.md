# Security Policy

## Reporting a vulnerability

To report a security vulnerability in the QuickBugs SDK, email:

**security@quickbugs.com**

Please do not open a public GitHub issue for security vulnerabilities.

We aim to respond within 48 hours and will keep you updated on the fix timeline.

## Scope

This policy covers the open source SDK packages:
- `packages/react`
- `packages/core`
- `packages/mcp`

For vulnerabilities in the QuickBugs platform (quickbugs.com), use the same email.
